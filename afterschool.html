MIGO 智能拼插课程

鼓励孩子们运用掌握的知识、技巧、方法等进行创新，独立创作出来自于自己创意的作品，并且所创造的作品都具有功能性、目的性及意义性。MIGO以“探索、尝试、创新、分享”为教育核心，通过产品主推儿童物联网电子模块科普工具（智能电子积木），启发儿童的STEAM思维。基于电子智能积木和教学系统，为小朋友提供能启发工程智能的工具型玩具。
电子模块根据不同功能和结构，划分为电源，输入，链接，输出四大模块。通过简单的模块串联，把精致的儿童玩具打造成一个充满声、光、电的动力等的缤纷世界。简单来说，对于培养儿童对电子科技知识的学习兴趣很有用。

Code Monkey 趣学趣玩桌游课程

Code Monkey诞生于以色列——一个非常重视科技教育的国家，它是一款在线编程教育课程，以闯关模式进行的编程游戏。目前，CodeMonkey已经推出了超过250个关卡，在全球范围内超过250万的用户，并且进入许多新式的学校里，包括硅谷著名的AltSchool。目前已有20个国家语言版本。Code Monkey采用编程教学语言CoffeeScript，它是JavaScript的一种转译语言，执行效率非常高。其代码指令简介，可读性强，非常接近自然语言，很适合作为变成教育的tt启蒙语言。Code Monkey采用编程教学语言CoffeeScript，它是JavaScript的一种转译语言，执行效率非常高。其代码指令简介，可读性强，非常接近自然语言，很适合作为变成教育的启蒙语言。

Game Zone 趣学趣玩桌游课程

K’Nex 构建进阶课程

K'Nex Education 是世界知名的搭建类玩具品牌，专为5-12岁学龄前及小学阶段儿童设计的教具产品，通过真实地搭建建筑模型，培养学生科学、技术、工程、数学方面的能力。

  • 动态教学

    激发学生积极参与学习，鼓励他们进 行科学实验、科学调查。

  • 替代真实建筑

    搭建过程能帮助学生思考真实世界中的建筑物如何构造，形成基本的建筑力学知识。

  • 教育标准

    与国家小学新课标的创客及STEM教育相适配，课程包含科技、技术、工程、数学等内容。

  • 以解决问题为根本出发点

    课程将通过抛出挑战的方式，让学生从了解问题、分析问题、设计解决方案、解决问题的流程进行。让学生拥有解决问题的能力。

  • 构建稳定、储存简单

    K’Nex的搭建和拆解都十分简单，构建好的模型将十分稳固。在一节课时内，面对简单的模型，搭建和拆解都可以完成，拆解后可轻松存放。
